# 🎲 Random File Tool

A modern Windows desktop application for randomly selecting images and videos from any folder.

The program can:

- randomly select files
- copy files
- directly create ZIP archives
- scan subfolders automatically
- run as a portable application without installation

---

# ✨ Features

✅ Modern user interface  
✅ Dark mode  
✅ Portable EXE  
✅ Random file selection  
✅ Image & video support  
✅ ZIP archive creation  
✅ Automatic subfolder scanning  
✅ Saved settings  
✅ Beginner friendly UI  

---

# ⚙️ Features in Detail

## 📂 Source Folder
Select the folder where random files should be taken from.

Subfolders are scanned automatically.

---

## 📁 Target Folder
Select the folder where copied files or ZIP archives should be saved.

---

## 🎲 Random Selection
The application randomly selects a user-defined number of files.

---

## 📦 ZIP Mode
Files can be directly packed into a ZIP archive.

---

## 💾 Saved Settings
The application automatically saves:

- Source folder
- Target folder
- Mode
- File amount
- Theme
